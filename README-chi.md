# COC-bot: 《部落冲突》自动化研究项目

## [英文](./README.md)  |   中文

### 1. 概述
**COC-bot** 是一个基于 **Airtest** 和 **OpenCV** 构建的《部落冲突》高级自动化框架。本项目作为一个游戏自动化的深度学习案例研究，主要关注：
- **状态机工作流**：在主世界和夜世界基地之间智能切换。
- **高级计算机视觉**：多模板匹配、基于置信度的世界检测和色彩空间分析。
- **OCR驱动的决策**：使用 `ddddocr` 进行实时战斗计时器和加载状态监控。

---

### 2. 核心功能

#### 🏠 主世界基地
- **自动收集**：一键收取金币、圣水和暗黑重油。
- **智能导航**：自动缩小镜头并寻路至夜世界飞船。
- **弹窗处理**：智能关闭活动新闻、商店优惠和"奖杯"奖励。

#### 🌙 夜世界基地
- **资源管理**：高效收集建造者金币和圣水。
- **高级训练系统**：支持清理军营和重新训练特定部队组合。
- **双阶段战斗逻辑**：支持新的BH10+两阶段战斗机制，包括增援部队部署。
- **可选战斗风格**：
  - 🧙 **女巫流（专注BH9）**：
    - *主阶段*：英雄 + 1巨人（坦克）+ 5女巫（召唤师）。
    - *增援*：1巨人 + 1女巫。
  - 🏹 **巨人弓箭手流（专注BH4）**：
    - *主阶段*：3巨人 + 1组弓箭手，用于精准的早期清理。

### 4. 架构与模块

#### 4.1 模块职责
- **`main.py`**：控制中心。处理CLI参数解析、世界调度和异常恢复。
- **`cocbot/common.py`**：硬件抽象层。管理ADB连接、`force_snapshot`（用于高分辨率稳定性）和OCR功能。
- **`cocbot/advanced.py`**：高级逻辑，包括`detect_world()`（基于置信度加权）和`双指捏合缩放`视图控制。
- **`cocbot/home.py`** 和 **`night.py`**：领域特定任务。`night.py`包含复杂的两阶段战斗状态机。
- **`cocbot/_assets.py`**：2K分辨率模板的集中库。

#### 4.2 运行时工作流
```text
[开始] -> [连接ADB] -> [启动游戏]
   |
   v
[检测世界] <-------------+
   |                         |
   +--[主世界]--> [收集] -> [前往夜世界] --+
   |                                           |
   +--[夜世界]--> [收集] -> [训练] -> [战斗] -> [返回主世界]
   |                                           |
[处理异常] <---------------------------+
```

### 5. 环境与设置

**先决条件：**
- **模拟器**：https://www.ldplayer.net/（推荐）。
- **分辨率**：**必须设置为 2560 x 1440 (DPI 480)**。
- **Python**：3.9+

**安装：**
```bash
# 推荐：Conda
conda env create -f environment.yml
conda activate COC-bot

# 关键依赖说明：
# 为避免Airtest和OCR之间的冲突，我们使用：
# opencv-contrib-python==4.6.0.66
```

### 6. 使用与CLI参数

#### 基本启动
```bash
python main.py --version tencent --night_faction witch
```

#### CLI参数表
| 参数              | 默认值          | 可选值              | 描述                                    |
| ----------------- | --------------- | ------------------- | --------------------------------------- |
| `--device`        | `emulator-5554` | -                   | ADB 设备序列号                          |
| `--version`       | `tencent`       | `tencent`, `global` | 游戏包版本（腾讯服/国际服）             |
| `--night_faction` | `witch`         | `witch`, `archer`   | 战斗策略（BH9女巫流 / BH4巨人弓箭手流） |
| `--loglevel`      | `0`             | `0, 1, 2`           | 0: 正常, 1: 流程, 2: 深度调试           |
| `--cap_method`    | `JAVACAP`       | `JAVACAP, ADBCAP`   | 截图方法（ADBCAP较慢但更稳定）          |

### 7. 技术说明
- **2K分辨率支持**：此脚本专门针对2560x1440分辨率调优。大多数素材在1080p下未经重新缩放将失效。
- **ADB版本管理**：项目包含强制同步环境和模拟器之间ADB版本的逻辑，以防止`设备连接超时`。
- **OCR区域扫描**：脚本对顶部中央的战斗计时器使用特定的关键区域扫描，以最小化CPU占用。
